# What's On My Disk?

> 一个能告诉你 **"硬盘里到底装了什么"** 的半自动磁盘资产盘点工具。

## 这个项目要解决的问题

WizTree 告诉你 `D:\xyz\` 占了 87 GB——但它**没法告诉你**这到底是 Steam 游戏、Python 虚拟环境、还是被遗忘的 4K 视频素材。你还得一个个文件夹打开来猜。

我们想做的,是把 **"看到陌生文件夹 → 猜它是干啥的 → 决定能不能删"** 这个最累的环节外包给 AI。

## 一句话定位

> **不是清理工具,是盘点工具。**
> 它不替你决定删什么,它帮你看清硬盘里到底是什么——剩下的事你自己拍板。
